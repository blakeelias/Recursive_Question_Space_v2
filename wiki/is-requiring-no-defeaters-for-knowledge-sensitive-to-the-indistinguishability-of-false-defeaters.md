# Is requiring no defeaters for knowledge sensitive to the indistinguishability of false defeaters?

**Node Type:** antithesis
**Node ID:** f9433d17-7ab1-47d3-8250-f7345d5539cd
**Depth:** 2
**Terminal:** No

**Parent:** [Defeasibility Theory](defeasibility-theory.md)

## Content

**It is possible to have false defeaters that appear legitimate but do not actually defeat the justification.**, **Determining whether a defeater is false or true is not always straightforward or possible with current information.**, **If knowledge excludes any defeaters, distinguishing false ones complicates the condition of having no true defeaters.**
